# XmlContactsExtractor

Extracts contact information from XML.

## Installation

Add this line to your application's Gemfile:

    gem 'xml_contacts_extractor'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install xml_contacts_extractor
